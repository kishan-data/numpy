# NumPy Basics Practice

This repository contains a Jupyter Notebook focused on learning and practicing the fundamentals of **NumPy**, one of the most important Python libraries for numerical computing and data science.

## 📌 Topics Covered

The notebook includes hands-on exercises on:

* Creating NumPy arrays
* Understanding array dimensions
* Array reshaping
* Indexing and slicing
* Boolean filtering
* Mathematical operations
* Trigonometric functions
* Sorting arrays
* Finding indices using `where`
* Linear algebra operations

  * Determinant
  * Inverse
  * Eigenvalues and Eigenvectors
* Dot product
* Transpose of arrays
